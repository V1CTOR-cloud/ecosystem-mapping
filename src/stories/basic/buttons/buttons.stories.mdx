# Buttons

Button component based on the Chakra-UI. We have added a few custom theme to have our own look.
We can pass all the props of the Chakra-UI Button component.

## Classic

<Canvas>
  <Story
    name="Classic"
    args={{
      text: "Classic Button"
    }}
  >
    {(args) => <Button>{args.text}</Button>}
  </Story>
</Canvas>

<ArgsTable story="Classic" />

The buttons under are all variant of the classic button.
