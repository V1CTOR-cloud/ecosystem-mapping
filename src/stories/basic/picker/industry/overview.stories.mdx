# Industry

Component that allows the user to pick an industry that includes a main industry and a sub-industry through a
predefined selection.

Depending on the case, we can allow the user to select multiple industry or not. If this is the case, then he can add
and remove as much as he wants.


## Multiple industry

The component displayed is a multiple industry selection, we can then select more than one industry.

### With no initials values

We do not have any initials values, we are in the case where we are in a creation mode in a form.
